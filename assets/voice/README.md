# Voice files

Put TTS-generated MP3 files in this folder with these exact names:

- `capture.mp3`: computer captures a piece
- `check.mp3`: computer gives check
- `capture-check.mp3`: computer captures and gives check
- `win.mp3`: computer wins
- `lose.mp3`: player wins

The app tries MP3 first. If an MP3 is missing or cannot be played, it falls back to the existing WAV placeholder with the same meaning.
